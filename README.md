## Ejercicio 1

### Pseudocódigo
Declara una función que acepte una palabra como parámetro y la devuelva con todos sus caracteres
en mayúsculas y separados por espacios.

Ejemplo: Para "Ana" devolverá "A N A".

```
Inicio
Leer palabra
  resultado = a mayúsculas(palabra)
  resultado = partir cada carácter(resultado)
  resultado = juntar con ‘ ’ (resultado)
  Escribir resultado
Fin
```
